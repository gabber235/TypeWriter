# Shortcuts
This page contains a list of keyboard shortcuts that you can use when using typewriter.\
But first what is your operating system?
<Tabs groupId="operating-systems">
  <TabItem value="windows" label="Windows">
| Shortcut                                                  | Description                                                                              |
| ----------------------------------------------------------| ---------------------------------------------------------------------------------------- |
| `ctrl + k `or` ctrl + space`                              | Open search box                                                                          |
| `tab `or` ctrl + n`                                       | Next item                                                                                |
| `shift + tab  `or` ctrl +  p`                             | Previous item                                                                            |
| `ctrl + shift + p`                                        | Publish pages                                                                            |
  </TabItem>
  <TabItem value="mac" label="macOS">
| Shortcut                                                  | Description                                                                              |
| ----------------------------------------------------------| ---------------------------------------------------------------------------------------- |
| `⌘ + k `or` ⌘ + space`                                  | Open search box                                                                          |
| `tab `or` ⌘ + n`                                         | Next item                                                                                |
| `shift + tab  `or` ⌘ +  p`                               | Previous item                                                                            |
| `⌘ + shift + p`                                          | Publish pages                                                                            |
  </TabItem>
</Tabs>

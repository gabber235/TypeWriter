# Znp Cs Plus Adapter

The ZNPCsPlus adapter allows you to create custom interactions with NPCs.

:::caution Untested
This adapter is untested. It may not work as expected. Please report any issues you find.
:::

## Entries

### Cinematics

| Name | Description |
| ---- | ----------- |
| [Reference Npc Cinematic](ZNPCsPlusAdapter/entries/cinematic/znpc_reference_npc_cinematic) | A reference to an existing npc specifically for cinematic |
| [Self Npc Cinematic](ZNPCsPlusAdapter/entries/cinematic/znpc_self_npc_cinematic) | The player itself as an cinematic npc |

### Entitys

| Name | Description |
| ---- | ----------- |
| [Reference Npc](ZNPCsPlusAdapter/entries/entity/znpc_reference_npc) | When the npc is not managed by TypeWriter |

### Events

| Name | Description |
| ---- | ----------- |
| [Npc Interact Event](ZNPCsPlusAdapter/entries/event/znpc_on_npc_interact) | When a player clicks on an NPC |

# Basic Adapter

The Basic Adapter contains all the essential entries for Typewriter.
In most cases, it should be installed with Typewriter.
If you haven't installed Typewriter or the adapter yet,
please follow the [Installation Guide](../../docs/02-installation-guide.md)
first.

## Entries

### Actions

| Name | Description |
| ---- | ----------- |
| [Add Potion Effect Action](BasicAdapter/entries/action/add_potion_effect) | Add a potion effect to the player |
| [Cinematic](BasicAdapter/entries/action/cinematic) | Start a new cinematic |
| [Console Command Action](BasicAdapter/entries/action/console_run_command) | Run command from console |
| [Delayed Action](BasicAdapter/entries/action/delayed_action) | Delay an action for a certain amount of time |
| [Drop Item Action](BasicAdapter/entries/action/drop_item) | Drop an item at location, or on player |
| [Give Item Action](BasicAdapter/entries/action/give_item) | Give an item to the player |
| [Message Action](BasicAdapter/entries/action/send_message) | Send a message to a player |
| [Play Sound Action](BasicAdapter/entries/action/play_sound) | Play sound at player, or location |
| [Player Command Action](BasicAdapter/entries/action/player_run_command) | Make player run command |
| [Random Trigger Gate](BasicAdapter/entries/action/random_trigger) | Randomly selects its connected triggers |
| [Remove Item Action](BasicAdapter/entries/action/remove_item) | Remove an item from the players inventory |
| [Set Block Action](BasicAdapter/entries/action/set_block) | Set a block at a location |
| [Set Item Action](BasicAdapter/entries/action/set_item) | Set an item in a specific slot |
| [Show Title Action](BasicAdapter/entries/action/show_title) | Show a title to a player |
| [Simple Action](BasicAdapter/entries/action/simple_action) | Simple action to modify facts |
| [Spawn Particle Action](BasicAdapter/entries/action/spawn_particles) | Spawn particles at location |
| [Stop Sound Action](BasicAdapter/entries/action/stop_sound) | Stop a or all sounds for a player |
| [Teleport Action](BasicAdapter/entries/action/teleport) | Teleport a player |

### Facts

| Name | Description |
| ---- | ----------- |
| [Cron Fact](BasicAdapter/entries/fact/cron_fact) | Saved until a specified date, like (0 0 * * 1) |
| [Inventory Item Count Fact](BasicAdapter/entries/fact/inventory_item_count_fact) | The amount of a specific item in the player's inventory |
| [Number Placeholder Fact](BasicAdapter/entries/fact/number_placeholder) | Computed Fact for a placeholder number |
| [Permanent Fact](BasicAdapter/entries/fact/permanent_fact) | Saved permanently, it never gets removed |
| [Session Fact](BasicAdapter/entries/fact/session_fact) | Saved until a player logouts of the server |
| [Timed Fact](BasicAdapter/entries/fact/timed_fact) | Saved for a specified duration, like 20 minutes |
| [Value Placeholder Fact](BasicAdapter/entries/fact/value_placeholder) | Fact for a placeholder value |

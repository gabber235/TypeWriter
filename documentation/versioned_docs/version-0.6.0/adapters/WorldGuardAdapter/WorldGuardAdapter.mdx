# World Guard Adapter

The WorldGuard Adapter allows you to create dialogue that is triggered by WorldGuard regions.

## Entries

### Audiences

| Name | Description |
| ---- | ----------- |
| [Region Audience](./entries/audience/region_audience.mdx) | Filter players based on if they are in a region |

### Events

| Name | Description |
| ---- | ----------- |
| [Enter Region Event](./entries/event/on_enter_region.mdx) | When a player enters a WorldGuard region |
| [Exit Region Event](./entries/event/on_exit_region.mdx) | When a player exits a WorldGuard region |

### Facts

| Name | Description |
| ---- | ----------- |
| [In Region Fact](./entries/fact/in_region_fact.mdx) | If the player is in a WorldGuard region |

### Groups

| Name | Description |
| ---- | ----------- |
| [Region Group](./entries/group/region_group.mdx) | All players grouped by WorldGuard regions |
